# Daftar Kebutuhan Data Website (Checklist untuk Klien)

Dokumen ini berisi daftar seluruh data, teks, dan aset visual yang diperlukan untuk menggantikan data *dummy* (buatan) di website agar sesuai dengan identitas dan profil perusahaan Anda.

Mohon untuk mengisi dan melengkapi data-data di bawah ini:

---

## 1. Identitas & Informasi Dasar (Pengaturan Umum)
Informasi ini akan ditampilkan di berbagai tempat seperti *Header*, *Footer*, dan halaman Kontak.

- [ ] **Nama Bisnis / Merek:** (Contoh: D'Moroy)
- [ ] **Tagline / Slogan:** (Contoh: Mahakarya Anyaman Serat Alam)
- [ ] **Logo Perusahaan:** (Format PNG resolusi tinggi dengan *background* transparan)
- [ ] **Nomor WhatsApp Aktif:** (Untuk menerima pesanan & pertanyaan, contoh: 081234567890)
- [ ] **Alamat Email Resmi:** (Contoh: info@dmoroy.com)
- [ ] **Alamat Lengkap Workshop / Toko:**
- [ ] **Jam Operasional:** (Contoh: Senin - Jumat, 08:00 - 17:00)
- [ ] **Link Media Sosial:**
  - Instagram: 
  - Facebook: 
  - TikTok: 
  - Lainnya (jika ada): 

## 2. Konten Halaman Beranda (Home / Hero Section)
Bagian paling atas yang pertama kali dilihat pengunjung saat membuka website.

- [ ] **Judul Utama (Hero Title):** (Contoh: Koleksi Tas Anyaman Premium)
- [ ] **Sub-judul (Hero Subtitle):** Deskripsi singkat 1-2 kalimat pendukung judul utama.
- [ ] **Video Latar Belakang (Opsional):** Link YouTube jika ingin menampilkan cuplikan video profil di beranda.
- [ ] **Gambar Banner / Slider:** Jika menggunakan gambar (bukan video), siapkan 1-3 gambar lebar beresolusi tinggi (rekomendasi: lanskap, minimal 1920x1080 pixel).
  - Siapkan juga *Teks Promo* (Judul & Deskripsi) untuk masing-masing gambar tersebut.

## 3. Katalog Produk (Produk yang Dijual)
Data mendetail untuk setiap barang yang akan dipajang. (Data ini **sangat direkomendasikan** untuk dibuatkan dalam tabel Microsoft Excel/Spreadsheet terpisah).

Untuk setiap **Kategori Produk** (misal: Tas Jinjing, Dompet, Aksesoris), mohon siapkan:
- [ ] **Daftar Kategori**

Untuk setiap **Produk**, mohon siapkan:
- [ ] **Nama Produk**
- [ ] **Kategori Produk**
- [ ] **Harga Jual** (Atau tulis "Sembunyikan/Hubungi Admin" jika harga ingin dirahasiakan).
- [ ] **Deskripsi Lengkap:** (Sebutkan bahan, ukuran/dimensi pxlxt, keunggulan, perawatan, dsb).
- [ ] **Lokasi Pembuatan:** (Contoh: Muaro Jambi, Indonesia).
- [ ] **Foto Produk:** (Bisa lebih dari 1 foto per produk. Rekomendasi: format persegi/1:1, *background* estetik/putih bersih).

## 4. Profil Toko & Tim (Halaman Tentang Kami)
Teks yang menceritakan cerita di balik merek untuk membangun kepercayaan pelanggan. Struktur ini akan dimasukkan ke halaman "Tentang Kami".

- [ ] **Foto Banner / Gambar Utama Profil:** (Gambar beresolusi tinggi yang merepresentasikan bisnis / proses produksi).
- [ ] **Sejarah Singkat / Cerita Bisnis (Story):** Siapkan narasi 2 hingga 3 paragraf yang menceritakan awal mula berdirinya bisnis dan harapan perusahaan. Termasuk juga satu kalimat puitis/motto utama.
- [ ] **Visi & Misi:**
  - Teks Visi Utama: 
  - Teks Misi Utama: (Sediakan minimal 3 poin penjabaran misi).
- [ ] **3 Prinsip / Nilai Utama (Our Principles):** 
  - Prinsip 1: (Judul & Deskripsi Singkat)
  - Prinsip 2: (Judul & Deskripsi Singkat)
  - Prinsip 3: (Judul & Deskripsi Singkat)
- [ ] **Data Anggota Tim / Pengrajin Utama (Opsional, untuk dimuat ke profil):**
  - Nama Lengkap:
  - Jabatan / Peran: (Contoh: Founder, Kepala Pengrajin)
  - Foto Wajah (Profil):

## 5. Artikel / Berita & Edukasi (Halaman Blog)
Jika perusahaan ingin menampilkan informasi tambahan seperti cara merawat produk, berita kegiatan, liputan media, atau edukasi budaya.

- [ ] **Daftar Kategori Artikel:** (Contoh: Tips & Trik, Berita, Event)
- [ ] **Konten Artikel Awal (1-3 Artikel Pertama):**
  - Judul Artikel
  - Isi Artikel Lengkap
  - Foto Kover (*Featured Image*) Artikel

---

### 💡 Catatan Tambahan
1. Kumpulkan semua file foto/gambar di dalam satu folder (misalnya di **Google Drive** atau *Flashdisk*), lalu beri nama file gambar yang jelas, contoh: `Produk_Tas_Selempang_Hitam.jpg` agar tidak tertukar.
2. Untuk teks yang panjang (seperti cerita sejarah, daftar misi, deskripsi seluruh produk atau artikel blog), sebaiknya diketik dan disusun di dalam **Microsoft Word, Excel, atau Google Docs** agar mudah kami proses ke dalam website.
